# 📦 PET魔法英语屋 - 应用分享指南

## 🎯 快速分享步骤

### 方法一：直接分享已构建的文件（推荐）

你已经构建好了 macOS 版本，可以直接分享：

**macOS 用户：**
- 文件位置：`dist/PET魔法英语屋-1.0.0-arm64.dmg`（Apple Silicon 芯片）
- 或：`dist/PET魔法英语屋-1.0.0.dmg`（Intel 芯片）

**分享方式：**
1. 将 DMG 文件上传到云盘（如百度网盘、阿里云盘、iCloud）
2. 或通过 AirDrop 直接发送给附近的 Mac 用户
3. 或通过邮件发送（文件较大，建议使用云盘链接）

**安装步骤（给接收者）：**
1. 下载 DMG 文件
2. 双击打开 DMG 文件
3. 将应用拖拽到"应用程序"文件夹
4. 如果提示"无法打开，因为来自身份不明的开发者"：
   - 打开"系统设置" → "隐私与安全性"
   - 找到"已阻止的应用"，点击"仍要打开
   - 或在终端运行：`sudo xattr -cr /Applications/PET魔法英语屋.app`

## 🔨 构建其他平台版本

### macOS 版本（你当前已构建）

```bash
npm run build:mac
```

构建完成后，在 `dist/` 目录下会生成：
- `PET魔法英语屋-1.0.0-arm64.dmg` - Apple Silicon (M1/M2/M3)
- `PET魔法英语屋-1.0.0.dmg` - Intel 芯片

### Windows 版本

**注意：** 需要在 Windows 系统或使用交叉编译工具

```bash
npm run build:win
```

构建完成后，在 `dist/` 目录下会生成：
- `PET魔法英语屋 Setup 1.0.0.exe` - Windows 安装程序

**分享给 Windows 用户：**
- 直接分享 `.exe` 文件
- 接收者双击运行安装程序即可

### Linux 版本

```bash
npm run build:linux
```

构建完成后，在 `dist/` 目录下会生成：
- `PET魔法英语屋-1.0.0.AppImage` - Linux 应用镜像

**分享给 Linux 用户：**
- 直接分享 `.AppImage` 文件
- 接收者需要给文件添加执行权限：`chmod +x PET魔法英语屋-1.0.0.AppImage`
- 然后双击运行

### 构建所有平台（需要相应环境）

```bash
npm run build:all
```

## 📤 推荐的分享方式

### 1. 云盘分享（最推荐）

**优点：** 文件大小不受限制，支持断点续传

**步骤：**
1. 上传 DMG/EXE/AppImage 文件到云盘
2. 创建分享链接
3. 发送链接给用户

**推荐云盘：**
- 百度网盘
- 阿里云盘
- 腾讯微云
- Google Drive（国际用户）
- Dropbox（国际用户）

### 2. 直接传输

**AirDrop（Mac 到 Mac）：**
- 选择文件 → 右键 → 共享 → AirDrop
- 或拖拽文件到 Finder 侧边栏的 AirDrop

**USB/移动硬盘：**
- 复制文件到移动存储设备
- 用户直接安装

### 3. 邮件分享

**注意：** 邮件附件通常有大小限制（如 25MB）
- 如果文件较大，建议使用云盘链接

## 📋 给用户的安装说明

### macOS 用户

1. **下载 DMG 文件**
   - 根据你的 Mac 芯片类型选择：
     - Apple Silicon (M1/M2/M3)：下载 `PET魔法英语屋-1.0.0-arm64.dmg`
     - Intel 芯片：下载 `PET魔法英语屋-1.0.0.dmg`

2. **打开 DMG 文件**
   - 双击下载的 DMG 文件

3. **安装应用**
   - 将应用图标拖拽到"应用程序"文件夹

4. **处理安全提示（如遇到）**
   
   **方法一（推荐）：**
   - 打开"系统设置" → "隐私与安全性"
   - 找到"已阻止的应用"
   - 点击"仍要打开"

   **方法二（终端）：**
   ```bash
   sudo xattr -cr /Applications/PET魔法英语屋.app
   ```

5. **运行应用**
   - 在"应用程序"文件夹中找到"PET魔法英语屋"
   - 双击启动

### Windows 用户

1. **下载安装程序**
   - 下载 `PET魔法英语屋 Setup 1.0.0.exe`

2. **运行安装程序**
   - 双击 `.exe` 文件
   - 按照安装向导完成安装

3. **启动应用**
   - 从开始菜单或桌面快捷方式启动

### Linux 用户

1. **下载 AppImage 文件**
   - 下载 `PET魔法英语屋-1.0.0.AppImage`

2. **添加执行权限**
   ```bash
   chmod +x PET魔法英语屋-1.0.0.AppImage
   ```

3. **运行应用**
   - 双击文件运行
   - 或命令行运行：`./PET魔法英语屋-1.0.0.AppImage`

## ⚠️ 注意事项

### 文件大小

- macOS DMG：约 150-200 MB
- Windows EXE：约 150-200 MB
- Linux AppImage：约 150-200 MB

### 系统要求

- **macOS：** macOS 10.13 或更高版本
- **Windows：** Windows 10 或更高版本
- **Linux：** 主流发行版（Ubuntu、Fedora、Debian 等）

### 数据存储

- 应用数据存储在本地（localStorage）
- 每个用户的数据是独立的
- 如需迁移数据，可以使用应用内的"导出数据"和"恢复数据"功能

### 更新应用

- 当前版本：1.0.0
- 如需更新，重新构建并分享新版本
- 建议在分享时告知用户版本号
