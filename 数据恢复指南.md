# PET英语屋数据恢复指南

## 🚨 问题：所有功能都不能用了，数据也没了

## 📋 快速解决方案

### 方法一：使用诊断工具（推荐）

1. 在浏览器中打开 `诊断工具.html`
2. 页面会自动运行诊断，显示当前系统状态
3. 如果发现问题，点击"恢复内置数据"按钮
4. 恢复成功后，打开 `src/index.html` 查看数据

### 方法二：使用快速恢复工具

1. 在浏览器中打开 `恢复数据-快速版.html`
2. 点击"1️⃣ 检查当前数据"查看当前状态
3. 点击"2️⃣ 恢复内置数据"恢复数据
4. 恢复成功后，打开 `src/index.html` 查看数据

### 方法三：使用完整恢复工具

1. 在浏览器中打开 `恢复数据.html`
2. 按照页面提示操作：
   - 检查当前数据状态
   - 恢复内置数据
   - 验证数据是否恢复成功

## 🔍 诊断工具功能说明

### 系统检查
- 检查内置数据是否加载
- 检查 localStorage 中是否有数据
- 检查浏览器兼容性
- 提供诊断建议

### 数据恢复
- 从内置数据文件恢复所有数据
- 自动保存到多个备份位置
- 显示恢复的数据统计

### 清除数据
- 清除所有 localStorage 中的 PET 相关数据
- 用于重置应用状态

## 📊 预期数据统计

恢复成功后，应该看到以下数据：

```
单词: 50 个
阅读: 20 题
固定搭配: 20 题
完形: 20 题
```

## ⚠️ 注意事项

1. **数据备份**：恢复操作会覆盖当前 localStorage 中的数据，请确保您真的需要恢复
2. **浏览器兼容性**：确保您的浏览器支持 localStorage 和 JSON
3. **文件路径**：确保 `src/pet-content.js` 文件存在且可访问

## 🛠️ 故障排除

### 问题1：诊断工具显示"内置数据未加载"

**原因**：`src/pet-content.js` 文件未正确加载

**解决方案**：
1. 检查 `src/pet-content.js` 文件是否存在
2. 确保文件路径正确
3. 检查浏览器控制台是否有错误信息

### 问题2：恢复后数据仍然为空

**原因**：localStorage 可能被禁用或已满

**解决方案**：
1. 检查浏览器是否启用了 localStorage
2. 尝试清除浏览器缓存
3. 尝试使用其他浏览器

### 问题3：恢复后某些功能仍然不能使用

**原因**：可能是代码逻辑问题

**解决方案**：
1. 打开浏览器开发者工具（F12）
2. 查看 Console 标签页是否有错误信息
3. 根据错误信息进行相应的修复

## 📞 获取帮助

如果以上方法都无法解决问题，请：

1. 打开 `诊断工具.html` 运行完整诊断
2. 截图保存诊断结果
3. 检查浏览器控制台的错误信息
4. 联系技术支持并提供诊断结果

## ✅ 恢复成功后的验证步骤

1. 打开 `src/index.html`
2. 查看页面底部是否有数据统计
3. 测试各个功能模块：
   - 单词背诵
   - 阅读理解
   - 固定搭配
   - 完形填空
4. 关闭浏览器后重新打开，确认数据是否保留

数据恢复成功后，您就可以正常使用 PET英语屋 进行学习了！
